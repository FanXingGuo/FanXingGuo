构造函数

  • 用于初始化对象成员属性
  • 构造函数函数名，必须与类名一致
  • 构造函数没有返回值
  • 同一个类，可以有多个参数列表不同的构造函数
  • 当且仅当类没有声明构造函数是，编译器会自动声明一个无参无内容构造函数

#include <iostream>
#include <string.h>
#include "../lib/Student.h"
using namespace std;


int main(int argc,char ** argv) {
    // 创建和销毁单个对象
    Student *student=new Student();
    delete student;
    //创建和销毁多个对象
    Student *students=new Student[5];// 此时会调用无参构造函数
    delete []students;
    return 0;
}

this指针

在对象调用成员函数时，会额外将本身地址传递给成员函数。在函数体内部，可以通过this指针，获取到编译器隐式传入当前对象的地址，并访问对象的成员。

    // 使用this 区分 类成员变量和参数
    strncpy(this->name,name,20);
}
