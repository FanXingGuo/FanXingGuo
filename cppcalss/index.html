C++ 对象

构造函数、析构函数

由于对象会在内存中占用一段空间，所以会有生命周期，也就是会有内存空间的申请和释放。

构造函数是对象申请内存空间之后自动调用的函数，析构函数是对象空间被销毁前自动调用的函数。

构造函数声明

构造函数没有返回值，构造函数函数名要有类名一致。可以有参数列表

// main.cpp
#include <iostream>
#include <string.h>
#include "Student.h"
using namespace std;


int main(int argc,char ** argv) {
    Student stu;
    // 修改成员属性
    strncpy(stu.name,"Tom",20);
    stu.score=100;

    stu.introduce();

    Student * pStu2; //使用指针
    pStu2=&stu;
    pStu2->score=99; // 分量运算符 访问成员变量
    pStu2->introduce();

    return 0;
}
